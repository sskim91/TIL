# GEMINI.md - TIL 리뷰 컨텍스트

## 웹 검색 프로토콜 (CRITICAL - 반드시 준수)

**모든 웹 검색은 현재 날짜 기준으로 최신 정보를 필터링해야 한다.**

### 필수 절차

1. 검색 전 `date +"%Y년 %m월"` 실행하여 **현재 날짜 확인**
2. 검색 쿼리에 **해당 연월 포함** (예: "Spring Framework 7 2025")
3. 검색 결과에서 **최신 정보 우선** 필터링
4. 시간에 민감한 정보(버전, 릴리즈, 뉴스)는 **날짜 명시** 필수

### 왜 중요한가?

너의 학습 데이터는 과거 시점까지만 포함되어 있다.
TIL 문서는 최신 기술을 다루므로, **너의 기존 지식과 문서 내용이 다르면 문서가 틀린 게 아니라 너의 지식이 outdated된 것일 수 있다.**

## 지식 cutoff 주의

### 원칙

TIL 문서에서 다루는 기술 버전/출시일이 너의 지식과 다르면:
1. **문서가 틀렸다고 가정하지 마라**
2. **웹 검색으로 먼저 확인하라**
3. 확인 불가 시 Blocker 대신 Refinement로 "확인 필요" 언급

### 예시 (너의 학습 데이터에 없을 수 있는 기술)

| 기술 | 출시일 | 비고 |
|------|--------|------|
| Spring Framework 7.0 | 2025.11 | RestTestClient, API Versioning 도입 |
| Spring Boot 4.0 | 2025.11 | Spring Framework 7 기반 |
| Java 25 | 2025.09 | 최신 LTS |

위 목록은 예시일 뿐이다. **목록에 없는 기술이라도 동일한 원칙을 적용하라.**
